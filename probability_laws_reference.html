I
Set Theory
Complement
A' = S − A
Difference
A − B = A ∩ B'
De Morgan 1
(A ∪ B)' = A' ∩ B'
De Morgan 2
(A ∩ B)' = A' ∪ B'
Complement laws
A ∪ A' = S | A ∩ A' = ∅ | (A')' = A
Identity laws
A ∪ ∅ = A | A ∩ S = A

II
Basic Axioms of Probability
Range
0 ≤ P(A) ≤ 1
Certain event
P(S) = 1
Impossible event
P(∅) = 0
Classical probability
P(E) = n(E) / n(S)
Complement rule
P(A') = 1 − P(A)
III
Addition Rules
General — two events
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
Mutually exclusive — A ∩ B = ∅
Two events
P(A ∪ B) = P(A) + P(B)
General — three events
P(A∪B∪C) = P(A)+P(B)+P(C) − P(A∩B) − P(A∩C) − P(B∩C) + P(A∩B∩C)
Mutually exclusive
Three events
P(A) + P(B) + P(C)
Independent
Three events
1 − P(A')·P(B')·P(C')
IV
Conditional Probability & Independence
P(A given B)
P(A|B) = P(A∩B) / P(B)
P(B given A)
P(B|A) = P(A∩B) / P(A)
Multiplication rule
P(A∩B) = P(A|B)·P(B)
Complement of condition
P(A|B') = P(A∩B') / P(B')
Independent events
If A and B are independent
P(A∩B) = P(A)×P(B) → P(A|B) = P(A) and P(B|A) = P(B)
V
Special Relationships
Only A (not B)
P(A∩B') = P(A) − P(A∩B)
Only B (not A)
P(A'∩B) = P(B) − P(A∩B)
Neither A nor B
P(A'∩B') = 1 − P(A∪B)
At most one of A, B
P(A'∪B') = 1 − P(A∩B)
Exactly one — A or B but not both
P(A∩B') + P(A'∩B) = P(A∪B) − P(A∩B)

A
Core Formulas
Definition
P(A|B) = P(A∩B) / P(B)
Symmetric form
P(B|A) = P(A∩B) / P(A)
Multiplication rule
P(A∩B) = P(A|B)·P(B) = P(B|A)·P(A)
P(B') = 1 − P(B)
P(A|B') = P(A∩B') / P(B')
Key identity — builds most solutions
P(A∩B') = P(A) − P(A∩B)
Example 1
Events C and D: P(C∩D')=0.1, P(C'∩D)=0.15, P(C'∩D')=0.2
P(C'∩D') = 0.2 → P(C∪D) = 0.8. The four Venn regions sum to 1: P(C∩D) = 0.8 − 0.1 − 0.15 = 0.15 — wait: rework from P(C)=0.65, P(D)=0.7 from the class solution.
P(C∪D) = 1 − P(C'∩D')
= 1 − 0.2 = 0.8
P(D') = 1 − P(D)
= 1 − 0.7 = 0.3
P(C∩D') given
= 0.1
P(C) from class
= 0.65
b(i) P(C|D') P(C∩D') / P(D') = 0.1 / 0.3 = 1/3 ≈ 0.333
b(ii) P(D'|C) P(D'∩C) / P(C) = 0.1 / 0.65 = 2/13 ≈ 0.154
Example 2 — Q9
Events E and F: P(E)=0.35, P(F)=0.45, P(E∩F')=0.15
Use P(E∩F') = P(E) − P(E∩F) to find the intersection first.
P(E∩F) = P(E) − P(E∩F')
= 0.35 − 0.15 = 0.20
P(F∩E') = P(F) − P(E∩F)
= 0.45 − 0.20 = 0.25
P(E') = 1 − P(E)
= 1 − 0.35 = 0.65
P(E∩F)
= 0.20 = 1/5
a) P(E∩F) 0.20 = 1/5
b(i) P(E|F) P(E∩F)/P(F) = 0.20/0.45 = 4/9
b(ii) P(F|E') P(F∩E')/P(E') = 0.25/0.65 = 5/13

A
Random Variable — Theory
Definition
X : S → ℝ (function on sample space)
P.d.f (discrete)
f(x) = P[X = x]

Discrete r.v.

Takes finite or countably infinite values (e.g. 0,1,2,3…)

Continuous r.v.

Takes values on an interval — uncountably infinite (e.g. all real x ≥ 0)

B
Discrete Probability Density Function (P.d.f)
Condition 1 — range
0 ≤ f(x) ≤ 1 for all x
Condition 2 — total probability
Σ f(xᵢ) = 1
How to find P.d.f
List all X values · count favourable outcomes · divide by total outcomes
C
Cumulative Distribution Function (C.D.F)
Definition — running total of P.d.f
F(x) = P(X ≤ x) = Σ f(xᵢ) for all xᵢ ≤ x
Boundary values
F(−∞) = 0 | F(+∞) = 1
Non-decreasing property
a < b → F(a) ≤ F(b)
Building the CDF
F(xₙ) = f(x₁) + f(x₂) + … + f(xₙ) (cumulative sum)
Ex 1
Coin tossed twice. X = number of heads. Find P.d.f and C.D.F.
S = {HH, HT, TH, TT} — 4 equally likely outcomes
x 0 (TT) 1 (HT or TH) 2 (HH)
f(x) 1/4 2/4 = 1/2 1/4
Σ f(x) = 1/4 + 2/4 + 1/4 = 1.0 ✓
F(x) Condition
0 −∞ < x < 0
1/4 0 ≤ x < 1
3/4 1 ≤ x < 2
1.0 2 ≤ x < ∞
